Из неопубликованной документации к объетку Promise

Объект Promise - это Гамлет в мире JS.

Он ожидает события.

А потом принимает решение, которое невозможно изменить.

Но режисеры, как и программисты, любят Гамлета/Promise не за это.

А за то, что спектакли по одной пьесе / экземпляры объекта Promise

могут идти один за другим!

      
экземпляр объекта Promise ждет события, чтобы выбрать:

      const promise =
      new Promise ((resolve, reject) => {
        if ('куплю билет') {
           resolve(театр("Гамлет"))
        }else{
           reject(сериал("Игра престолов"))
        }
      }

А вот инструкция "Что делать" на любой выбор:

          promise.then(
            data => "Гамлет: интриги, дуэль, яд",
            error => "Игра престолов: интриги, дуэль, яд"
          );

У объекта Promise есть два основных метода:

Promise.resolve() - возвращает обещанное, например значение или объект.

Promise.reject() - возвращает объект Promise, который был отклонен по указанной причине.

Даже среди категоричных promise метод Promise.all(list) выделялся.

list = [promise1, promise1, ..., promiseN]


Он соглашался вернуть resolve только если все promise в его списке также вернули resolve!


Даже одного reject от кого-то из list было достаточно и для его reject!


И он нетерпелив! Как только получит первый reject - все, остальные могут не отвечать.

Proof: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Promise/all

Метод fetch на смену XMLHttpRequest

есть полифиллы (и не один)

Очень простой синтаксис


          // url (required), options (optional)

               fetch('https://goit.ua', {
                 method: 'get'
               })then(function(response) {

               }).catch(function(err) {
	            // Error :(
            });

Если добавить return, то получим цепочку обработки событий

        // url (required), options (optional)
             fetch('https://goit.ua', {
               method: 'get'
             }).then(function(response) {
                ... return anyValue;
              }).then(function (anyValue){
                 ...
              })
             }).catch(function(err) {
            // Error :(
          });
