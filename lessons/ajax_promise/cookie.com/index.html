Зачем хранить данные клиента?
  1. Идентификация сессии
  2. Браузерные игры
  3. Высоконагруженные порталы
  4. Образовательные приложения (тесты)
  5. Автозаполнение

Какие существуют системы хранения данных?

  1. Cookie: поддержка всеми браузерами 100%. Недостаток: отключается пользователем
  2. Web Strorage: поддержка современными браузерами 94,75%. Рекомендуемое к использованию
  3. Indexed DB: поддержка современными браузерами ~84% кроме OperaMini
  4. Web SQL: поддержка современными браузерами 81%. Недостаток: не поддерживается (и не будут) FF, Edge, IE, OperaMini

Update: document.cookie = "user_name=Bart Simpson";

Delete: document.cookie = "user_name=Bart;max-age=-1";
